# eventlensrepo
EventLens is a no-code Excel event-study model measuring how 15 major market shocks (2001–2024) moved 8 US benchmarks — from the S&amp;P 500 to WTI crude and the VIX. 120 studies, 118,000+ live formulas, Brown–Warner significance tests, and every price validated against an independent event database. No code, no add-ins: open it and it runs.
